``psiturk_status`` command
==========================


Usage
------

::

   psiturk_status

Display startup screen with message from `psiturk.org <http://psiturk.org>`__.


Example
-------

::


   [psiTurk server:off mode:sdbx #HITs:1]$ psiturk_status
